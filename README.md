# ML Research Agent (RAG)

This repository contains both a Next.js web application and a FastAPI/Streamlit backend for querying Machine Learning research papers using Retrieval-Augmented Generation (RAG).

## Hugging Face Spaces Deployment

This repository is configured to deploy directly to Hugging Face Spaces using the Streamlit SDK.

The main application file is located at `rag-arxiv/app/streamlit_app.py`.
